Weather App

Weather App is a sleek and interactive ReactJS-based application that provides real-time weather updates for any searched location. Built using ReactJS, CSS, and the OpenWeatherMap API, the app fetches live temperature data and dynamically adjusts the background to match the weather conditions. If the temperature is 15°C or below, a cold-themed background is displayed; for temperatures between 15°C and 25°C, a warm background appears; and for temperatures above 25°C, an autumn-themed image is shown. Along with the temperature, the app also provides a brief weather description, ensuring a visually appealing and informative user experience.

Tap Music

Tap Music is a fun and interactive web app built using HTML, CSS, and JavaScript, designed to create simple tunes through button taps. When a button at the bottom of the screen is tapped, a corresponding sound plays, and for a visually engaging experience, a small colored circle moves in sync with the duration of the sound. By tapping different buttons in sequence, users can create their own rhythmic patterns and melodies. The app combines sound and motion to deliver an enjoyable and dynamic musical interaction.
